# 🐕 Criadero WOBYK — Sistema de Gestión Genealógica

Sistema web completo para gestión de criadero de **Fila Brasileiro**, con árbol genealógico, simulación de cruzas y control sanitario.

## 🔐 Acceso por defecto
- **Usuario:** `wobyk`
- **Contraseña:** `wobyk2024`

> Cambie las credenciales desde el panel de **Configuración** después del primer ingreso.

## ✨ Funcionalidades

| Módulo | Descripción |
|---|---|
| **Panel** | Resumen general, alertas de vacunas, reproductores activos |
| **Plantel** | Registro completo de ejemplares con foto, genealogía y estado |
| **Camadas** | Historial de cruzas y nacimientos |
| **Vacunas** | Control sanitario con alertas automáticas (≤60 días) |
| **Simulación** | Cálculo de Coeficiente de Consanguinidad (COI) — Método Wright |
| **Árbol Genealógico** | Pedigree visual de 2 a 8 generaciones |
| **Configuración** | Datos del criadero, usuario, contraseña |

## 🚀 Uso en GitHub Pages

1. Subir la carpeta a un repositorio GitHub
2. Ir a **Settings → Pages**
3. Source: `Deploy from branch → main → / (root)`
4. El sitio estará disponible en `https://tuusuario.github.io/nombre-repo`

## 💾 Almacenamiento

Los datos se guardan en `localStorage` del navegador. No requiere servidor ni base de datos.
